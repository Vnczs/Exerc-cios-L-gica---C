# Exerc-cios-L-gica---C
Trabalho de Lógica de programação na linguagem C
Este repositório reúne uma série de algoritmos desenvolvidos para consolidar conceitos fundamentais de lógica matemática e manipulação de dados em baixo nível, componentes essenciais da graduação em Analise e Desenvolvimento de Sistemas

 Conceitos Aplicados:

1. Lógica Proposicional e Tabelas Verdade
Implementação de algoritmos que simulam o raciocínio lógico matemático:

Validação de proposições lógicas através de estruturas condicionais (if/else).

Automação de resultados baseados em entradas booleanas.

2. Conversão de Bases Numéricas
Desenvolvimento de funções para converter valores entre diferentes sistemas de numeração, fundamental para entender a comunicação entre software e hardware:

Decimal para Binário: Lógica de divisões sucessivas e restos.

3. Estruturas de Controle de Fluxo
Uso avançado de laços de repetição (for, while) para processamento de dados e validação de algoritmos matemáticos complexos.

 Tecnologias Utilizadas
Linguagem: C (Padrão C99)
